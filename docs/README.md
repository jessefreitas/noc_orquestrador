# Docs Index

Este diretorio concentra a documentacao do projeto em tres trilhas:

- `docs/active/`: documentos ativos e canonicos usados no dia a dia.
- `docs/archive/`: material legado/consultivo que nao deve poluir a raiz.
- `docs/policies/`: politicas operacionais de governanca do workspace.

Arquivos de migracao e reorganizacao ficam na raiz de `docs/`, por exemplo:

- `docs/MIGRATION_2026-02-23.md`

Regra de ouro: nao deletar historico sem aprovacao. Se nao estiver em uso, mover para `docs/archive/`.
